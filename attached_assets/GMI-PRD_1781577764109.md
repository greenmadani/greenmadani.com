## 1. Project Overview

### 1.1 Company Background

Green Madani International Private Ltd. (GMI) is a diversified business group, previously known as Green Universe Group. The company is headquartered in Dhaka, Bangladesh, and focuses on agriculture solutions, consumer goods, hospitality, education, healthcare, and media — operating across 12 subsidiary brands.

### 1.2 Project Goal

Build a premium, internationally-positioned corporate website that:
- Establishes GMI as a trustworthy, innovative, and sustainable organization
- Attracts customers, farmers, distributors, dealers, business partners, and investors
- Serves as the digital headquarters for all 12 GMI subsidiaries
- Supports lead generation, partnership inquiries, and career applications

### 1.3 Website Type

Multi-page corporate group website with CMS-powered admin panel — comparable in scale and quality to international conglomerate websites (e.g., Tata Group, PRAN-RFL, Akij Group).

## 6. Design System

### 6.1 Components

| Component | Description |
|---|---|
| Navbar | Sticky, transparent-to-solid, mobile hamburger |
| Hero Banner | Full-width, dark green, pattern overlay |
| Section Label | Small uppercase gold text above headings |
| Business Card | Image + badge + name + desc + CTA |
| Product Card | Image + category + name + desc + button |
| Blog Card | Image + tag + date + title + excerpt |
| Stat Counter | Large number + label, animated on scroll |
| CTA Banner | Full-width dark section, centered, button |
| Footer | 4-column, dark green, Islamic pattern bg |
| Form | Clean white card, labeled inputs, gold submit |
| Modal | Product detail overlay |
| Breadcrumb | Small gray nav below page hero |
| Toast | Success/error form submission feedback |

### 6.2 Button Styles

| Type | Background | Text | Border |
|---|---|---|---|
| Primary | `#C8960C` gold | `#1A1A1A` dark | none |
| Secondary | `#1A5C38` green | `#FFFFFF` white | none |
| Outline Light | transparent | `#FFFFFF` white | 1.5px white |
| Outline Dark | transparent | `#1A5C38` green | 1.5px green |
| Ghost | transparent | `#C8960C` gold | none |

All buttons: `border-radius: 6px`, `padding: 12px 28px`, `font-weight: 600`, hover: darken 10% + subtle scale(1.02)

### 6.3 Spacing System

- Base unit: 8px
- Section padding: 80px top/bottom (desktop), 48px (mobile)
- Card gap: 24px
- Component internal padding: 24–32px

### 6.4 Animation

| Element | Animation |
|---|---|
| Hero text | Fade up on load, 0.6s ease |
| Section headings | Fade up on scroll (Intersection Observer) |
| Cards | Stagger fade-up, 100ms delay each |
| Stat counters | Count-up animation on scroll |
| Navbar | Smooth background transition on scroll |
| Hover (cards) | translateY(-4px), box-shadow |
| Hover (buttons) | scale(1.02), background darken |
| Page transitions | Fade in/out |

### 7.3 Database Schema

**Tables:**
- `products` (id, name, category, description, image, status, created_at)
- `categories` (id, name, slug, type)
- `businesses` (id, name, slug, description, image, order, status)
- `news` (id, title, slug, content, image, category, author, published_at, status)
- `careers` (id, title, department, location, type, description, status, posted_at)
- `applications` (id, job_id, name, email, phone, cover_letter, cv_path, created_at)
- `inquiries` (id, name, email, phone, subject, message, created_at, status)
- `settings` (key, value)

### 7.4 SEO Requirements

- Semantic HTML5 tags (header, main, section, article, footer)
- Meta title + description per page
- Open Graph tags (og:title, og:description, og:image)
- Twitter Card meta tags
- JSON-LD Schema markup:
  - Organization schema
  - LocalBusiness schema
  - BreadcrumbList schema
  - Product schema (product pages)
  - Article schema (news pages)
- `sitemap.xml` — auto-generated, includes all pages
- `robots.txt` — allow all, sitemap reference
- Canonical URLs
- Alt text on all images
- Heading hierarchy (H1 → H2 → H3)
- Page load < 3 seconds
- Core Web Vitals compliant

### 7.5 Performance

- Images: WebP format, lazy loading (`loading="lazy"`)
- CSS/JS: minified for production
- Google Fonts: preconnect + font-display: swap
- Critical CSS inlined
- Browser caching via `.htaccess`
- GZIP compression enabled

### 7.6 Responsiveness

| Breakpoint | Width | Layout |
|---|---|---|
| Mobile | < 640px | 1 column, hamburger menu |
| Tablet | 640–1024px | 2 columns, condensed nav |
| Desktop | > 1024px | Full layout |
| Wide | > 1280px | Max-width 1280px container |

### 7.7 Accessibility

- WCAG 2.1 AA compliant
- Keyboard navigable
- Focus rings visible
- ARIA labels on icons and interactive elements
- `prefers-reduced-motion` respected for animations
- Sufficient color contrast ratios
